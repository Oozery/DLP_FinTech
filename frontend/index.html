Why This Project Migrated to ECC

🔐
Real security at 256-bit

DLP at 256-bit gives only ~40-bit security — breakable. ECC at 256-bit gives 128-bit security — the same level as Bitcoin.

⚡
Faster ZK proofs

Scalar multiplication on 256-bit numbers is faster than modular exponentiation on 3072-bit numbers needed for equivalent DLP security.

📐
Same Schnorr protocol

The ZK proof structure is identical — only the group operation changed from gx mod p to x·P. All security properties are preserved.

Edge Case

Large Balance Chunking

What happens when a balance exceeds the curve order n? We decompose it into base-n chunks.

The Problem

x ≥ n → x·P wraps around

If balance (x) exceeds curve order n, scalar multiplication wraps and the proof breaks. Solution: base-n decomposition.

✗
Without chunking: x ≥ n causes x·P = (x mod n)·P, proof breaks
✓
With chunking: x = c₀ + c₁·n + c₂·n² + ..., each cᵢ < n, proven via Qi = cᵢ·P
✓
Combined commitment: Q = Q₀ + Q₁ + Q₂ + ... (EC point addition — homomorphic)
