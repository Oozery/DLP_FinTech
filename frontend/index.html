Edge Case

Balance Beyond DLP Range

What happens when x exceeds the subgroup order q? We chunk it.

The Problem

x ≥ q → gx mod p wraps around

If balance (x) exceeds q, modular arithmetic breaks the proof. Solution: base-q decomposition.

✗
Without chunking: x ≥ q causes gx = g(x mod q), proof breaks
✓
With chunking: x = c₀ + c₁·q + c₂·q² + ..., each cᵢ < q
